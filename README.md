# FutureAsset GitHub Website Kit — Version 3

This version updates the full logo system.

## Main change
The light-background logo now uses:
- Deep Ink `#0A2025`
- Green Accent `#6FA529`

It no longer uses light grey.

## Included
- responsive website
- dark-background logo
- light-background logo
- monochrome logo
- dark and light mark-only versions
- favicon
- brand guide
- website copy reference

## Before publishing
Open `index.html` and replace:

`YOUR-EMAIL-HERE`

with your preferred email address.

## Publish with GitHub Pages
1. Unzip this package.
2. Upload all files and folders to your GitHub repository.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/root`.
6. Save.

## Replacing the previous package
Delete or overwrite the old files in your GitHub repository with the files in this package. Keep the folder structure unchanged.
